# Time Manager — Frontend

> Time Manager 的前端 Flutter 项目。

## 构建

```bash
flutter build web
```

## 部署

```bash
python -m http.server 8080 -d build/web
```

前端默认运行在 <http://localhost:8080>。

## 技术栈

- Flutter + Riverpod
- GoRouter 路由
- Material 3 设计
- Dio HTTP 客户端
- `table_calendar` 日历组件
